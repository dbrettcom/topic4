IoT Applications

  • Applications that connect devices so that they can communicate with each other without human intereference
  • Features include sensors, devices, applications, user interfaces, servers, cloud, etc
  • Examples include smart buildings, security systems, inventory management, transportation and agricuture

Smart Buildings

  • Smart buildings collect data about the users of the building to optimise and automate the building's functions
  • Features include light and temperature sensors, cameras, meters, processing systems, power sources, etc
  • Examples include temperature control, light management, hazard detection, energy optimisation, etc

IoT Interfaces

  • IoT interfaces need to show the status of sensor data and present it in a way that is intuitive for the user
  • IoT interfaces need inputs (eg. sensors), processing, outputs (eg. actuators) and memory to function
  • Users of these interaces include the people who live and work in the building, as well as the corporations who run them

Examples:

Temperature Control Automated Lighting Traffic Monitoring
Automated Manufacturing Smart Grids Fleet Management
Digital Kiosks Smart Watches Smart Scales
